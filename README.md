## 项目介绍

#### 主要特性

- 使用最新技术栈，社区资源丰富。
- 支持数据字典，可方便地对一些状态进行管理
- 支持接口级别的功能权限与数据权限，可自定义操作
- 自定义权限注解与匿名接口注解，可快速对接口拦截与放行
- 前后端统一异常拦截处理，统一输出异常，避免繁琐的判断
